# ml_techarena_huawei

Решение задачи ML TechArena Siberian Challenge 2024 от Huawei.

## Описание задачи

Задача заключается в реализации эффективного индекса для поиска ближайших соседей (k-NN) в больших векторных базах данных. Проект поддерживает датасеты SIFT и GIST, используя структуры данных вроде KD-Tree и методы снижения размерности PCA.

## Структура проекта

- `build_index.py`: Скрипт для построения индекса на основе обучающих данных.
- `test_py.py`: Основной скрипт для выполнения поиска ближайших соседей по запросу.
- `index_build`: Bash-скрипт для запуска построения индекса.
- `index_scan`: Bash-скрипт для запуска поиска.
- `Makefile`: Файл с целями для сборки и запуска.

## Установка и зависимости

Проект использует Python 3 и следующие библиотеки:
- numpy
- struct (стандартная библиотека)
- os, sys (стандартные)
- pickle (стандартная)

Установите зависимости с помощью pip:
```
pip install numpy
```

## Запуск

### Построение индекса

Используйте Makefile для построения индекса:
```
make do_index INDEX_ARGS="--data=путь_к_данным"
```

Или напрямую:
```
./index_build --data=путь_к_данным
```

### Выполнение поиска

```
make do_scan SCAN_ARGS="--query=запрос"
```

Или напрямую:
```
./index_scan --query=запрос
```

Где `запрос` - вектор в формате [x1,x2,...,xn] или строка без скобок.

## Формат данных

- Датасеты в формате .fvecs (binary format для векторов).
- Поддерживаемые датасеты: SIFT (sift_base.fvecs) и GIST (gist_base.fvecs).
- Пути к датасетам: ../datasets/sift/ и ../datasets/gist/

## Алгоритмы

- **KD-Tree**: Для поиска ближайших соседей.
- **PCA**: Для снижения размерности векторов.
- Евклидово расстояние для измерения близости.

## Пример использования

1. Постройте индекс:
   ```
   ./index_build --data=../datasets/sift/sift_base.fvecs
   ```

2. Выполните поиск:
   ```
   ./index_scan --query="1.0 2.0 3.0 ... 128.0"
   ```

## Тестирование

Запустите `python3 test_py.py` с параметрами для тестирования функциональности.
